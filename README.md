## How does it work?
* Enter your twitter profile URL in the input text field.
* Download your threads as CSV.

|Thread URL | Tweet | Date | Favorites | Retweets | 
| --------- | ----- | ---- | --------- | -------- |

* Choose a thread  from the extracted ones to display on the app.
